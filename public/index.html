🎊 Chào Mừng VIP

Phát video/âm thanh khi user vào phòng LIVE hoặc tặng quà — theo TikTok ID hoặc theo cấp độ.
📁 Nhóm hồ sơ
Cài hiệu ứng riêng cho từng TikTok ID — 7 trigger: Vào phòng / Táp tim / Follow / Share / Comment / Bao lì xì / Lên cấp.
📡 Cách phát hiện user vào phòng (theo thứ tự ưu tiên):
① Top contributors list (~30s/lần, TỰ ĐỘNG) — bắt được user top trong phòng KHÔNG cần tương tác. Đây là cách CHÍNH.
② Member event — TikTok tự gửi cho viewer "đáng kể" vào phòng.
③ ❤️ Táp tim / Comment / Follow — fallback nếu user không trong top: chỉ cần táp 1 cái tim → fire ngay.
🔁 Vào lại phòng: sau khi user vắng mặt 30 giây (không có signal), nếu họ quay lại sẽ fire effect LẠI. Ở liên tục trong phòng → KHÔNG re-fire (tránh spam).
⚠ Giới hạn TikTok: User level rất thấp + lurk hoàn toàn + không trong top → không phát hiện được (giới hạn protobuf TikTok).
Trigger cho BẤT KỲ user nào trong phòng LIVE. Đặt Min Level đủ cao để tránh spam.
Server quản lý hàng đợi — overlay nhận từng item theo nhịp perItemMinMs.
